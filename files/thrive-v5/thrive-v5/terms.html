Terms & Participation Guidelines

Ground rules for safe, respectful, and transparent use of THRIVE website and programme features.

Last updated: June 2026.

Website use

Use the website lawfully and respectfully. Do not attempt to bypass admin controls, submit malicious content, scrape private information, or interfere with programme operations.

Quizzes and leaderboards

Quiz and leaderboard features are intended for programme engagement. Display names should not include offensive language, impersonation, or sensitive information. Scores are self-reported and unverified in the current static implementation.

Media and uploads

Only authorised administrators may upload programme media. Photos or videos of minors should be published only with appropriate consent and must be removed promptly if a valid takedown request is received.

Donations

Donation details should be verified with THRIVE before transfer. Donors may request anonymous acknowledgement. Private donor information must not be placed on public pages or shared without explicit consent.

Admin changes

Browser-based admin tools are suitable for content previews and low-risk updates only. Updates that affect donations, personal data, or participant records should use a secure backend with authentication, roles, audit logs, and backups.

Contact

Questions about these guidelines can be sent to hello@thrivefoundation.org.
