Terms & Participation Guidelines

Ground rules for safe, respectful, and transparent use of TriveFoundation website and programme features.

Last updated: 14 June 2026. These terms should be reviewed before production launch.

Website use

Use the website lawfully and respectfully. Do not attempt to bypass admin controls, submit malicious content, scrape private information, or interfere with programme operations.

Quizzes and leaderboards

Quiz and leaderboard features are intended for programme engagement. Display names should not include private information, offensive language, impersonation, or school-sensitive data unless authorised.

Media and uploads

Only authorised administrators should upload programme media. Photos or videos of minors should be published only with appropriate consent and should be removed promptly if a valid takedown request is received.

Donations

Donation details should be verified with TriveFoundation before transfer. Donors may request anonymous acknowledgement, and private donor information must not be placed on public pages.

Admin changes

Browser-based admin tools are suitable only for low-risk content previews. Production updates that affect donations, personal data, or participant records should be handled through a secure backend with authentication, roles, audit logs, and backups.

Contact

Questions about these guidelines can be sent to hello@trivefoundation.org.
